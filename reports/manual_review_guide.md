# Manual Review Guide

Reviewers should score each generated case using the CSV template exported by the evaluation script.

## Scoring rubric

- `visual_credibility`
  - 1: clearly unrealistic
  - 3: plausible but weak
  - 5: highly credible
- `nasal_naturalness`
  - 1: anatomically implausible
  - 3: mixed realism
  - 5: natural-looking change
- `artifact_severity`
  - 1: severe artifacts
  - 3: moderate artifacts
  - 5: negligible artifacts

## Notes

- Focus on the nasal region and adjacent profile contour.
- Ignore minor lighting shifts if the shape remains consistent.
- Flag any identity drift or geometry collapse in the notes field.
